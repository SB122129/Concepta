# Concepta Demo Script (5-7 minutes)

## 1. Problem setup (30-45s)
- "Students in intro thermodynamics often confuse entropy, spontaneity, and heat vs temperature."
- "Concepta focuses on this exact learner wall and provides evidence-backed diagnostics and remediation."

## 2. Learning flow (60-90s)
- Open Learning tab.
- Enter a prompt like: "Explain why a spontaneous reaction can still be slow."
- Show Explanation, Visual, Simulation, and Verification tabs.
- In Verification, point out the Evidence Panel and source categories.

## 3. Diagnostic assessment (90-120s)
- Go to Test tab.
- Start a Pretest with a selected topic.
- On each question, set confidence (1-5) and answer.
- Intentionally miss one question.
- Show the mistake analysis + targeted remediation card with linked resources.

## 4. Assessment report (60-90s)
- Finish quiz and show Assessment Report.
- Highlight:
  - accuracy
  - misconceptions encountered
  - strengths
  - confidence calibration and confidence gap
  - recommended next step

## 5. Impact dashboard (60-90s)
- Save and open Metrics.
- Show:
  - pretest/posttest measured gain
  - topic-level gains
  - attempt timeline (accuracy vs confidence)
- Click Export JSON and Export CSV.

## 6. Closing (20-30s)
- "Concepta does not only generate explanations; it measures and improves learner understanding with evidence-backed intervention."
